****PASOS PARA CREAR UNA CARPETA***

    1. Abrimos el Git Bash
    2. Digitamos el comando "PWD"
    3. Digitar "mkdir"
    4. Digitar "ls"
    5. Digitar cd

.****PASOS PARA CREAR UN ARCHIVO***

    1. Ingresar a la carpeta por medio del comando "cd" donde vamos a crear los archivos
    2. Digitar el comando "touch"

    ****PASOS PARA GENERAR UNA LLAVE SEGURA EN SSH****

      3. Copiamos el codigo o la llave que se genera en Bash y la pegamos en Git Hub.

    ****PASOS PARA CLONAR****

      1. Crear Primero el Repositorio en Git Hub
      2. Clonar el Repositorio en Git Hub
      3. Copiar el enlace y Pegarlo en Bash
      4. Ingresar a Bash y digitar "Git Clone" y lue+go pegar el enlace
      5. Digitar "Git Status"
      1. Git add -A: Prepara y selecciona todos los Archivos que han sido Modificados.
      2. Git commit -m: "Descripción"
      3. Git Push:Empuja todo el código con todo é imágenes.
